# Contribuir a AWS Cost Optimizer AI

¡Gracias por tu interés en contribuir! Este proyecto está abierto a mejoras y sugerencias.

## 🤝 Cómo contribuir

### 1. Fork del repositorio
\`\`\`bash
# Hacer fork en GitHub y clonar
git clone https://github.com/tu-usuario/aws-cost-optimizer-ai.git
cd aws-cost-optimizer-ai
\`\`\`

### 2. Crear rama para tu feature
\`\`\`bash
git checkout -b feature/mi-nueva-funcionalidad
\`\`\`

### 3. Hacer cambios y commit
\`\`\`bash
git add .
git commit -m "feat: descripción de la funcionalidad"
\`\`\`

### 4. Push y Pull Request
\`\`\`bash
git push origin feature/mi-nueva-funcionalidad
# Crear Pull Request en GitHub
\`\`\`

## 📝 Guía de estilo

- Código Python: PEP 8
- Docstrings: Google style
- Commits: Conventional Commits
- Tests: pytest

## 🧪 Tests

Asegúrate de que todos los tests pasen:
\`\`\`bash
pytest tests/ -v
\`\`\`
